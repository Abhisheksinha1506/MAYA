# Complete Maya Programming - Sample Files Archive

## Overview
This repository contains the complete sample code and project files for **"Complete Maya Programming"** Volumes 1 and 2, the definitive technical references for Autodesk Maya API programming. These books and their accompanying source code represent the authoritative guide to extending Maya through C++ plugins, MEL scripting, and Python integration.

## Project Details
- **Source**: "Complete Maya Programming" by David A.D. Gould
- **Platform**: Autodesk Maya (3D animation and modeling software)
- **Languages**: C++, MEL (Maya Embedded Language), Python
- **Volumes**: Vol 1 (Fundamentals), Vol 2 (Advanced Techniques)

## Technology Stack
- **Maya API**: C++ SDK for creating native Maya plugins
- **MEL**: Maya's proprietary scripting language
- **Python API**: Python bindings for Maya automation
- **OpenMaya**: Core Maya API library for geometry and scene manipulation

## Repository Structure

### Volume 1 - Fundamentals (119 files)
Covers the essential concepts of Maya programming:
- **Basic Plugin Architecture**: Node creation, command development
- **Geometry Manipulation**: Mesh editing, deformers, and modifiers
- **Scene Graph**: Understanding Maya's DAG (Directed Acyclic Graph)
- **MEL Scripting**: Procedural modeling and automation
- **Custom Tools**: UI development and shelf tools

### Volume 2 - Advanced Techniques (141 files)
Explores sophisticated Maya programming:
- **Advanced Deformers**: Custom skinning and rigging solutions
- **Dynamics Integration**: Particle systems and physics
- **Rendering Plugins**: Shader development and render passes
- **Performance Optimization**: Efficient data structures and caching
- **Production Pipelines**: Asset management and batch processing

## Key Technical Features
- **Plugin Development**: Complete examples of Maya plugin creation from initialization to deployment
- **API Patterns**: Best practices for working with Maya's complex object model
- **Cross-Platform Code**: Windows, macOS, and Linux compatibility
- **Production-Ready**: Real-world examples used in professional VFX and animation studios

## Historical Significance
"Complete Maya Programming" is considered the bible of Maya technical development. The sample code has educated generations of technical artists and pipeline developers at major studios including Pixar, ILM, and DreamWorks. These files represent the foundation of countless production tools and workflows in the film and game industries.
